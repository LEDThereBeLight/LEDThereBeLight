Linked Lists

A linked list is like a conga line, where every item holds on to the next one in the list. The two most common types of linked lists are singly linked lists. A singly linked list is one where both hands are on the person in front of you, and a doubly linked list is one where one hand is on the person in front of you, and one on the person behind you.

Singly Linked List

Singly linked lists are built from pairs of pointers. The first pointer points to an item, and the second pointer points to the next pair in the list.

Note how the second part of the last pair points to undefined, which tells us we're at the end of the list.

Doubly Linked List

Doubly linked lists are like singly linked lists, but they have an extra pointer to the previous item in the list.

Making a list

Let's make a singly linked list, the most common type of linked list.

Lists are built from nodes using the cons function, which is short for "construct."

function cons (head, tail) {
  return {
    head,
    tail
  }
}

We can construct lists of any length:

let EMPTY = undefined
let list = cons(1,
                cons(2,
                     cons(3,
                          cons(4,
                               cons(5, 
                                    EMPTY)))))
// list = { head: 1, tail:
//          { head: 2, tail: 
//            { head: 3, tail: 
//              { head: 4, tail: 
//                { head: 5, tail: 
//                  undefined }}}}}

We can build lists directly from other data structures, too. Here's how we might make a list from an array of items.

let array = [1, 2, 3, 4, 5]

let linkedList = array.reduceRight(
  (acc, curr) => cons(curr, acc), 
  EMPTY
)
