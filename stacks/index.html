Stacks

A stack is like a stack of pancakes. They are a Last In, First Out (LIFO) container, which means the first item to come out is the item that was most recently added.

Stacks have at least two methods:

  1. push: O(1), add an item
  2. pop: O(1), remove an item

But they may include additional methods:

  1. isEmpty: O(1), returns true if no items are in the stack
  2. peek: O(1), returns the item at the top of the stack without removing it

Implementation

Stacks are usually implemented with either arrays or linked lists.

Object Oriented

We'll create a stack object that owns all of the methods which work on it. We can store all of the items stored in the stack in a single array.

Here's the full, working code.

function Stack () {
  let size = 0
  let items = []
  
  let push = (item) => {
    size += 1
    items[size] = item
  }
  
  let pop = () => {
    if (size > 0) {
      let item = items[size]
      items[size] = undefined
      size -= 1
      return item
    }
  }
  
  let peek = () => items[size]
  
  let isEmpty = () => size === 0
  
  return {
    size: () => size,
    push,
    pop,
    peek,
    isEmpty
  }
}

Functional

Functional stacks don't wrap up all of their methods into a stack object. Instead, we write functions that take stacks and return new stacks.

let Stack = (size, items) => {
  return {
    size,
    items
  }
}
let isEmpty = (stack) => stack.size === 0
let size = (stack) => stack.size
let items = (stack) => stack.items

let push = (stack, item) => 
  Stack(size(stack) + 1, 
        cons(item, 
             items(stack))

let peek = (stack) => {
  if (isEmpty(stack)) 
    return console.log('Error - peek on empty stack')
  else return head(items(stack))
}

let pop = (stack) => {
  if (isEmpty(stack)) 
    return console.log('Error - pop on empty stack')
  else return cons(head(items(stack)), 
                   Stack(size(stack) - 1, 
                         tail(items(stack))))
}
